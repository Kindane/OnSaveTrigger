# On Save Trigger

Суть этого проекта заключается в слежке за модификацией файла, как только выбраный файл был модифицирован и сохранён то выполняется указаная вами команда.

## Installation

```LINUX ONLY!!!```
Для компиляции этого чуда вам нужно сделать пару простых шагов, а именно: <br>
1. Если у вас не установлен CMAKE, то я настоятельно рекоммендую вам это сделать, ибо ~~нехуй вам без него на линуксе быть~~ так будет проще собрать данный проект.<br>
2. У данного проекта вроде  нет никаких зависимостей, но он использует `inotify`, который вроде как изначально стоит в линуксе, но я не уверен.
3. Открыть терминал, перейти в директорию с этим проектом и ввести эту команду: $**mkdir build && cd build; cmake .. && make;** <br>
Если у вас, по какому-то странному стечению обстоятельств не вылезло ни единой ошибки и компиляция прошла успешно, то я вас ПОЗДРАВЛЯЮ, вы успешно скомпилировали данный проект.

## Starting

Перед запуском проекта вам нужно будет его скомпилировать!!.<br>
Вам нужно из терминала перейти в папку 'build' и ввести в терминал данную команду: $**./OnSave ИМЯ_ФАЙЛА "КОМАНДА_ДЛЯ_ВЫПОЛНЕНИЯ"**<br> поздравляю, вы успешно запустили скрипт,
теперь он активно будет следить за модификацией файла, и если заметит, что файл был изменён, то выполнит КОМАНДА_ДЛЯ_ВЫПОЛНЕНИЯ.
